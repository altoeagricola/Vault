## Levidian Ltd - LOOP Technology

**Evolução:** Levidian é evolução/rebranding de Cambridge Nanosystems/CamGraph (aliases históricos). A empresa continua ativa com novo posicionamento: conversão de metano em grafeno via tecnologia LOOP proprietária.

## Modelo de Negócio - Tese Dupla

**Fato:** Tecnologia LOOP modular para conversão pré-combustão de metano em grafeno + hidrogênio limpo. Instalação de piloto ADNOC/Baker Hughes em Habshan (jan/2025).

**Claim:** Escalabilidade 50.000 t/ano grafeno via rede de dispositivos até 2030.

**Modelo comercial:** Venda de (1) tecnologia LOOP para operadores de óleo/gás + (2) grafeno como coproducto valorizado.

## Tecnologia LOOP

| Atributo | Informação | Classificação |
|---|---|---|
| **Operação** | Modular, catalisador-free, water-free | Fato técnico |
| **Condições** | Baixa temperatura/pressão | Fato |
| **Entrada** | Metano (pré-combustão) | Fato |
| **Saídas** | Grafeno + H₂ (coproducto valioso) | Fato |
| **Versão** | LOOP Gen 2 anunciada (2024) para escala industrial | Claim |

## Escala e Marcos

- **Piloto ADNOC (jan/2025):** >1 t/ano grafeno + >1 t/ano H₂ (operação incipiente)
- **Ambição 2030:** 50.000 t/ano grafeno (tipicamente 50x de multiplicador = 15+ anos com sucesso)
- **Parcerias:** ADNOC, Baker Hughes, Dana Gas (anúncio)

## Implicação para MGgrafeno

### ✅ Benchmark CONCEITUAL (Não de Escala)

**Lição de Levidian:** Acoplar grafeno a uma dor ambiental/regulatória (descarbonização upstream) é estratégia válida de diferenciação. Para MGgrafeno, oportunidades análogas:

- **Economia circular de resíduos:** Casca de palmáceas, resíduos agroindustriais (⚠️ cuidado com reputação palm)
- **Redução de clínquer em concreto:** Rota technical validável com benchmarks de resistência
- **Filtração/adsorção ambiental:** Modelo Directa Plus + Setcar (ambiental)
- **Biodegradabilidade/ciclo:** Agrofilmes com propriedades de degradação controlada

**Pré-requisito:** Haver rota técnica **independentemente validável**, não especulação de ESG.

### ❌ Benchmark NEGATIVO (Para Escala Imediata)

- NÃO é referência para capacidade comercial verificada
- Timeline 2030 é muito distante
- Aguardar evolução piloto ADNOC 2026+ antes de qualquer benchmark comparativo
- Não usar projeção de 50K t/ano em nenhum modelo de negócio sem revisão independente
