# Parceiros

Fichas CRM dos parceiros do ecossistema ALTOE Agricola — parceiros técnicos, ICTs, fornecedores e bancos com relacionamento ativo.

Cada nota representa uma organização parceira, seguindo o template `[[T-CRM]]`.

## Tipos de parceiros

| Tipo | Exemplos |
|------|----------|
| Parceiro técnico | IFES, LUMA, centros de pesquisa |
| ICT | Institutos de Ciência e Tecnologia |
| Fornecedor | Insumos, equipamentos, serviços |
| Banco / financiador | Linhas de crédito, fomento |

## Template

Usar `[[T-CRM]]` para criar novas fichas. Salvar o arquivo com o nome da organização (`Nome da Organização.md`).
