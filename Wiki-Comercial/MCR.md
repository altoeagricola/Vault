# Manual de Crédito Rural (MCR)

Manual oficial do Banco Central que define todas as regras do crédito rural no Brasil. Base normativa para todas as operações de crédito rural da [[ALTOE Agricola]].

---

## Linhas Principais Operadas pela ALTOE Agricola

### Pronaf Custeio

- Financia custos do ciclo produtivo: insumos, mão de obra, colheita
- Linha mais comum em [[Marilandia]] para pequenos produtores de conilon
- [[Sicoob]] tem se consolidado como banco mais flexível para esta linha

### Pronaf Investimento (Mais Alimentos / Moderfrota / Proirriga)

- Financia compra de equipamentos, máquinas, implementos, infraestrutura
- Exige Projeto de Crédito assinado por Responsável Técnico (RT)
- Nota: verificar habilitação específica de Rodrigo (CRQ) para assinar projetos de crédito rural
- ALTOE Agricola ainda não operou esta linha

### Pronaf Habitação Rural

- Financia construção, reforma ou ampliação de moradia rural
- Caso em operação: [[Pronaf Habitacao - Sergio Jose Altoe]] (via [[Cresol]])

### Linhas não-Pronaf (Moderfrota, Proirriga, Renovagro)

- Para produtores com renda acima do teto Pronaf (não possuem DAP/CAF)
- Condições e exigências diferentes
- ALTOE Agricola ainda não teve contato com essas linhas

## Impedimentos Comuns em Marilândia

| Impedimento | Dificuldade de Resolver |
|-------------|------------------------|
| CAR não regularizado | Fácil |
| CCIR não quitado | Fácil |
| ITR pendente | Fácil |
| Outorga de água vencida | Fácil |
| Licenciamento de secadores | Possível |
| CPF negativado | Caso a caso |
| Inadimplência bancária | Avaliar renegociação vs. nome sujo |
| Renda acima do teto Pronaf | Verificar declaração de renda |

**Procedimento quando há impedimento:** A ALTOE Agricola tenta resolver para o produtor. Exemplo: outorga vencida → informar o produtor e cobrar pelo serviço de regularização.

---

## Classificação de Finalidades

O crédito rural deve ser estritamente classificado entre:
- **Custeio** — despesas normais do ciclo produtivo
- **Investimento** — bens e serviços de longo prazo
- **Comercialização** — escoamento e comercialização da produção
- **Industrialização** — beneficiamento e processamento

## Renegociação

Dificuldades no fluxo de caixa por perdas acumuladas de safras anteriores são aceitas como motivo para renegociação de dívidas.
